Responsible for:

- Creating tables
- CRUD operations (Create, Read, Update, Delete)

#### models.py
Contains:
- Transaction model
- Data validation logic
- Object-oriented representations of records

#### utils.py
Provides utility functions such as:
- Date formatting
- Currency formatting
- CSV export
- Input validation
- Common helper methods

#### dashboard.py
Displays:
- Total Income
- Total Expenses
- Current Balance
- Recent Transactions
- Financial Overview
